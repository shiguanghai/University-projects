## CrossPlatform-newsApp

### 跨平台新闻App（React Native+Wechat）

- 基于网易头条新闻API的新闻首页
- 包含新闻列表（支持刷新获取数据）、轮播图和新闻详情页面
- 基于Mac M1芯片的 ReactNative 环境的ios端和微信小程序端
